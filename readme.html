# eCommerce Perfume Website This project is a feature-rich eCommerce perfume website developed using WordPress, WAMP, and MySQL. It was created during my internship at Sheefra as my first solo project for a client. The website is designed to provide a seamless shopping experience with efficient product management. ## Features - **Product Categorization**: Perfumes are categorized for Men, Women, and Unisex. - **Detailed Product Descriptions**: Each product includes a detailed description, pricing, and images. - **WooCommerce Integration**: Implemented WooCommerce for managing products, shopping carts, and orders. - **User-Friendly Interface**: Designed to be intuitive and easy to navigate for a smooth user experience. ## Technologies Used - **WordPress**: Content management system used to build the website. - **WAMP**: Windows, Apache, MySQL, and PHP stack used for local development. - **MySQL**: Database management system to store and manage product data. - **WooCommerce**: Plugin for eCommerce functionality. ## Hosted Link The project is hosted and can be accessed at [https://fragrance.sheefra.company/](https://fragrance.sheefra.company/). ## License This project is licensed under the MIT License.
